ActiveMQ Dashboard has been installed.

Access the dashboard:

  kubectl port-forward svc/{{ include "activemq-dashboard.frontend.serviceName" . }} 3000:80 -n {{ .Release.Namespace }}
  open http://localhost:3000

{{- if .Values.ingress.enabled }}

Ingress is enabled. The dashboard will be available at:
  http{{ if .Values.ingress.tls }}s{{ end }}://{{ .Values.ingress.host }}

{{- else }}

To enable ingress:
  helm upgrade {{ .Release.Name }} activemq-dashboard/activemq-dashboard \
    --reuse-values \
    --set ingress.enabled=true \
    --set ingress.host=activemq-dashboard.example.com

{{- end }}

Credentials are stored in Secret: {{ include "activemq-dashboard.secretName" . }}

To check pod status:
  kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/instance={{ .Release.Name }}

To check backend health:
  kubectl exec -n {{ .Release.Namespace }} deploy/{{ include "activemq-dashboard.fullname" . }}-backend -- \
    wget -qO- http://localhost:8080/api/v1/health
